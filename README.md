## 🎯 Objetivo do Projeto
Este repositório contém a análise estrutural e exploratória de um dataset de vendas da Amazon. O foco é identificar como as características dos produtos (preço, categoria e descontos) influenciam a satisfação do cliente (ratings) e o volume de vendas no marketplace.

## 🌍 Conexão com ODS (UNESCO)
O projeto está fundamentado na **ODS 12: Consumo e Produção Responsáveis**.
**Justificativa:** Através do estudo de padrões de consumo e feedback dos usuários, buscamos gerar insights que otimizem a oferta de produtos. Entender o que o consumidor realmente valoriza ajuda a reduzir o ciclo de devoluções e desperdícios logísticos, promovendo um ecossistema de e-commerce mais eficiente e sustentável.

## 📊 Estrutura de Dados
Diferente de grandes bases externas, este projeto hospeda sua base de dados principal diretamente no repositório para facilitar a reprodutibilidade:
* **Arquivo:** `data/amazon.csv`
* **Fonte:** [Amazon Sales Dataset - Kaggle](https://www.kaggle.com/datasets/karkavelrajaj/amazon-sales-dataset)
* **Conteúdo:** Informações sobre ID do produto, nome, categoria, preços (com e sem desconto), porcentagem de desconto, rating e quantidade de avaliações.

* **Bibliotecas Planejadas:** Pandas, Matplotlib, Seaborn.
